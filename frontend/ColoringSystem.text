WazaranPI Color System: Use bg-[#F3FFFC] for main page backgrounds, bg-[#DDFBF3] for header, sidebar, and footer, and bg-white for cards, modals, and containers; primary color is teal (bg-teal-500, text-teal-600) for buttons, icons, and active elements, with light primary (bg-teal-100, text-teal-700) for icon backgrounds and soft highlights; secondary accent is cyan (bg-cyan-500, text-cyan-600) for alternative actions; danger uses red (bg-red-500, text-red-600, bg-red-100) for delete and warnings; text should be text-slate-900 for main, text-slate-500 for secondary, and text-white or text-white/70 on colored backgrounds; borders use border-cyan-100; general rule: clickable elements use teal, backgrounds use white or soft green, text uses slate, and danger actions use red.
